# Online QEMU CLI Generator

This website is a tool to build QEMU command line graphically in just a few clicks:

1. [Open the website](https://willnode.github.io/qemu-config/)
2. Adjust few parameters, particularly the disk file name.
3. Change "Your Operating System" with the OS you're about to run the QEMU on
4. Copy the generated command line to a terminal that contain the disk to emulate
5. If it works, press "Save Configuration" to save it into your browser localStorage.
